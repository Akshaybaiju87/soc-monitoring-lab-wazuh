SOC Monitoring Lab using Wazuh

This project demonstrates a personal Security Operations Center (SOC) lab built to practice cybersecurity monitoring, threat detection, and incident response.

Tools Used:

-Wazuh (SIEM platform)
-ELK Stack (Elasticsearch, Logstash, Kibana)
-Kali Linux (attack simulation)
-Linux endpoint system


Project Overview:

In this lab environment, Wazuh agents were deployed on Linux endpoints to collect and monitor system logs. Security events were analyzed using the ELK Stack dashboards.

The lab simulates real SOC workflows including attack detection, log analysis, and incident response.

 Key Activities:

- Monitoring authentication logs from Linux systems
- Detecting SSH authentication failures
- Simulating attack activity using Kali Linux
- Investigating alerts generated by Wazuh
- Identifying suspicious IP addresses
- Blocking malicious IP addresses using firewall rules

 Example Security Events Detected

- SSH brute-force attempts
- Multiple authentication failures
- Suspicious login activity
